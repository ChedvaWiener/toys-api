Get started

     API Endpoint
 
         https://toys-api-service.onrender.com

By logging in you can add, edit, and remove your toys, Admin can delete and modify users and their toys.

All HTTP methods are supported. You can use http or https for your requests.

Sign Up


 Body example
 {
     "firstName": "Chedva",
     "lastName": "Wiener",
     "email": "merola.we@gmail.com",
     "password": "1Aa123456",
     "role": "user"
 }

Sign up a new user.
POST call to the following url :
https://toys-api-service.onrender.com/user/ 



 Result example :
 
 {
     "firstName": "Chedva",
     "lastName": "Wiener",
     "email": "merola.we@gmail.com",
     "password": "******",
     "date_created": "2023-11-30T17:57:11.789Z",
     "toys": [],
     "role": "user",
     "_id": "6568cf4140bc65ca52afa59f",
     "__v": 0
 }

BODY PARAMETERS

Name Required Description
firstName Yes (String) First name, min length 2 characters max length 100 characters.
lastName Yes (String) Last name, min length 2 characters max length 100 characters.
email Yes (Email) Email, min length 2 characters max length 100 characters.
password Yes (String) Password, min length 6 characters max length 100 characters. Must contain at least one lowercase letter, one uppercase letter, and one digit.
role No (Default: user) Role, possible values: "user", "admin".

Add a new toy.
POST call to the following url :
https://toys-api-service.onrender.com/user/


HEADERS KEY

Key Value
x-api-key Token (the token is valid for sixty minutes). 

 Result example :
 
 { 
     "name": "Baby Doctor",
     "produced_country": "China",
     "info": "Barbie Careers Doll & Playset",
     "category": "Dolls",
     "img_url": "no image",
     "price": 300,
     "date_created": "2023-12-02T18:18:05.051Z",
     "_id": "656b8431dcb540bde3501d22",
     "__v": 0
 }

BODY PARAMETERS

Name Required Description
name Yes (String) Name of the toy, min length 2 characters max length 100 characters.
produced_country Yes The country where the toy is made, min length 2 characters max length 50 characters.
info No (Default "No info") Additional information, min length 3 characters max length 1000 characters.
category Yes The toy category, min length 3 characters max length 50 characters.
img_url No (Default "No image") Image url, max length 500 characters.
price Yes (Number) The toy price, min value: 0.5, max value: MAX_SAFE_INTEGER.

Errors

The API uses the following error codes:

Error Code Meaning
400 Invalid body. This error appears if you miss required fields, exceed the limits, or add fields in the body request.
401 Password and email mismatch. This error appears if you Trying to log in with an incorrect email or password.
402 Unknown or unvalid user token. This error appears if you use an unknow user token or if the user token expired.
403 Permission denied. This error appears when a user tries to delete or edit a toy that is not theirs.
500 General error.
